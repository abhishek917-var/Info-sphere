# InfoSphere – DecodeLabs Project 1

InfoSphere is a front‑end web project built as part of DecodeLabs Project 1, designed to act as **“The Ultimate Encyclopedia for Everything.”**  
It focuses on presenting technology, trading, AI, coding, and other knowledge in a clean, searchable interface using a hero section, search bar, category filter, and curated information cards.

## 1. Overview

InfoSphere is a static website that serves as a centralized knowledge hub where users can **explore, learn, and grow with absolute information** across multiple domains.  
The homepage includes a strong headline (*“The Ultimate Encyclopedia for Everything”*), a search bar, a category dropdown, and rich content about technology, trading, AI tools, and global news.

---

## 2. Features

### Hero section with clear branding

- Headline: **“The Ultimate Encyclopedia for Everything”**  
- Supporting text: *“Explore, Learn, and Grow with Absolute Information, From Tech to Trade: Insights on Everything That Matters.”*

### Search bar with category dropdown

- Search input for user queries.  
- Category select box with: **All Categories, Technology, Trading, AI, Coding**.  
- Helps users narrow down the information they are interested in.

### Curated news & information cards

Section labeled **“Information”** containing multiple cards such as:

- **Google Meta** – news and updates about Google and Meta.  
- **JavaScript projects / Asian market cues / Stocks to watch** – focused on trading and market data.  
- **Guided learning** cards describing detailed topics like:
  - REC–PFC merger approval.  
  - NASA Swift telescope mission.  
  - Belkin 14‑in‑1 Thunderbolt 5 Dock.  
  - Mumbai EV and ethanol mobility transition.  
  - AI tools like Claude and Gemini.  
  - Trading basics and tools like Loom.

Each card includes short descriptive text and a **“Learn More”** or **“Read It”** link pointing to external information.

### Favorites & daily content concept

Text sections that describe:

- **Favorite Articles** – keeping your favorite articles at your fingertips, always available.  
- **The Best Content, handpicked for you in the everyday** – curated, editor‑style selection of top articles and updates so users don’t need to scroll endlessly.

## 4. How to Run

### Option A – Directly in Browser

1. Clone or download the repository:

   ```bash
   git clone https://github.com/your-username/infosphere-decodlabs-project1.git
   cd infosphere-decodlabs-project1
   ```

2. Find `project.html` in the folder.  
3. Double‑click `project.html` to open it in Chrome, Edge, or any browser.

### Option B – VS Code + Live Server

1. Open the project folder in **Visual Studio Code**.  
2. Install the **Live Server** extension (if not already installed).  
3. Right‑click `project.html` → **Open with Live Server**.  
4. The site will open on a local URL like `http://127.0.0.1:3000/project.html`.

## 5. Usage

- Type a topic (e.g. “AI”, “Trading”, “NASA”) in the search bar on the hero section.  
- Select a category from the dropdown (**Technology, Trading, AI, Coding**) to focus the information.  
- Scroll to the **“Information”** and **“Guided learning”** cards to read short summaries of each topic.  
- Click **“Read It”** or **“Learn More”** links to open detailed articles and news in your browser.
