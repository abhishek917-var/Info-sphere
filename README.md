InfoSphere – DecodeLabs Project 1
InfoSphere is a front‑end web project built as part of DecodeLabs Project 1, designed to act as “The Ultimate Encyclopedia for Everything.”
It focuses on presenting technology, trading, AI, coding and other knowledge in a clean, searchable interface.

2. Features
*Hero section with clear branding
Headline: “The Ultimate Encyclopedia for Everything” with supporting text explaining that users can explore topics from tech to trade and beyond.

*Search bar with category dropdown
An input field and a category select box (All Categories, Technology, Trading, AI, Coding) help users narrow down the information they are interested in.

*Information cards
Section labeled “Information” containing multiple cards like “Google Meta”, “Guided learning for personal assistance”, “Daily best content for everyone”, and domain‑specific content.

*Domain‑specific insights
Cards and text blocks covering:

Stock market, trading cues, merger updates.

NASA missions and space exploration.

Thunderbolt dock devices, EV mobility, and industry news.

Major AI tools and models like Claude and Gemini.

*Favorites & daily content concept
Descriptions for “Favorite articles” and “Daily best content” that explain how users can keep important articles handy and receive curated content.

*Contact section
Footer area showing InfoSphere’s location (Aligarh, Uttar Pradesh, India), email, and phone number for communication.

3. Tech Stack
*Adjust if needed according to your actual files:
HTML5 – Structure of all sections (navigation, hero, search, information cards, contact).
CSS3 – Layout, color scheme, typography, and responsiveness (if implemented).

4. Usage
Use the search bar on the hero section to type a topic you want to know about (e.g., “AI”, “Trading”, “NASA”).
Select a category from the dropdown (Technology, Trading, AI, Coding) to focus on that domain.
Scroll to the “Information” section to read cards that summarize news, insights, and curated content; click “Read It” / “Learn More” links to go to full articles if they are connected.
